STRISK-NOFA

Predicting need for surgery in small bowel obstruction

About the Tool

STRISK-NOFA is a free tool developed using multicentre surgical data to predict the risk of strangulation or non-operative treatment failure for patients with SBO. Built using the R Shiny framework.
